64-bit Version Available

The 64-bit Windows version of Amzi! is available and ready for download at the current 9-5-2 release level. Users with current licenses can download and use either version, or both. Current site license customers can use either version.

As always, we encourage institutes of learning to purchase our education site licenses to provide access to the full Amzi! Prolog + Logic Server for all students and faculty.

Amzi! Multiple Platform Support

Amzi! Prolog has an architecture similar to Java's. Amzi! compiles to a machine independent binary form that can be run on any Amzi! runtime. Currently, only the Windows versions of Amzi! are publicly available for download, but Amzi! has been ported to many different environments. If you are interested in running Amzi! on a platform other than Windows, please contact us.
