Mac OS X Version

Amzi! Prolog + Logic Server with the full Eclipse IDE is now available for OS X 64-bit Macs: Downloads

Linux Runtime

The Amzi! Prolog runtime is now running on CentOS64 which is binary compatible with Red Hat Enterprise Linux (RHEL). It will run both Amzi! Prolog and ARulesXL distributable applications. If you are interested in running your Amzi! applications on this platform, or others, please contact us.

Faster Downloads

Amzi! downloads should now be much faster, and we've removed the Captcha. Let us know if you have any problems with the new downloads.
