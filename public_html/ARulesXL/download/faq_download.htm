Download

The download failed, what do I do now?

You can try a different method to download, FTP instead of HTTP, or vice versa. If you use an FTP client to download, you can automatically resume the download from where you left off if you are disconnected or the transfer ends early.

You can also try one of the non-Amzi! sites (like Simtel.net and Winsite.com which has many worldwide mirrors).

Also you can order a CDROM and have it shipped anywhere in the world from either us or Winsite.com.

Licensing

I want to transfer my license (version 1.1.14 or later) to another Windows PC, how do I do that?

You need to unregister your current ARulesXL license key to transfer it to another PC. To do this, run the License Manager from the ARulesXL program menu and press the 'Remove License Key' button. This will provide you with a reference number. Write it down.

On the new PC, run the License Manager from the ARulesXL program menu and press the 'Get License Key' button. Enter the reference number, and the same user name and email you used on the original PC.

I want to transfer my license (prior to version 1.1.14) to another Windows PC, how do I do that?

You need to unregister your current ARulesXL license key to transfer it to another PC. To do this, run the License Manager from the ARulesXL program menu and press the 'Unregister/Transfer License' button.

This will guide you through the process and give you the unregister code to enter into the Transfer License web form.

I lost my unregister code.

Look in the root directory of your disk for a file named:

amzi_license_uninstall.txt (version 1.1.14 or later) or amzi_license_transfer.txt (before version 1.1.14).

It contains a record of your unregister code, reference number, user name and removed license key.

I need to re-install my Windows operating system, what do I do about my license key?

You need to unregister your ARulesXL license before re-installing your operating system. See the instructions above to do this.
