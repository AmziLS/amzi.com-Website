Requirements

ARulesXL runs under Microsoft Excel 2002 or later.

Installing ARulesXL

To install ARulesXL into Excel, double-click on the arulesxl_N-N.exe distribution file to run it. The installation program will prompt you with where to put the documentation, license manager and sample files (usually in Program Files\amzi\ARulesXL).

The ARulesXL Add-In will be installed in the Microsoft Office library directory for the current version of Excel on your PC. For Excel 2003, this is \Program Files\Microsoft Office\OFFICE11\Library. For Excel 2002, this is \Program Files\Microsoft Office\OFFICE10\Library.

In other languages 'Library' has a different name as follows:

Danish: BIBLTEK
Dutch: BIBLIO
Finnish: Kirjasto
French: Macrolib
German: MAKRO
Italian: Libreria
Norwegian: BIBL
Portuguese: BIBLIOTE
Spanish: MACROS
Swedish: MAKROBIB

The installation program will enable the ARulesXL Add-In for you. You can disable it via the Add-In Manager in Excel.

If you want to install the Add-In into multiple versions of Excel, copy the ARulesXL Add-In files to the library directory for each one. These files are:

ar_mysql.lsx
ar_osutils.lsx
arules.cfg
arules.dll
ARulesXL.xla
arules.xpl

Licensing

Activating Your License

ARulesXL is licensed to a single PC via a hardware fingerprint. Each PC has a different fingerprint. The fingerprint changes when you re-install your Windows operating system.

To activate your license, start the License Manager from the ARulesXL programs menu.

  • If you received a license key via email, under 'Install License Key' enter the username and license key you received by email after your purchase and press the 'Install License Key' button.
  • If you did NOT receive a license key, under 'Get License Key from Web' enter your reference number, user name and email address from your purchase receipt and click the 'Get License Key' button.

Transferring Your License

You can transfer your license to a new PC by using the ARulesXL License Manager and the www.ARulesXL.com Web site.

To do this, start the License Manager on your original PC. Your current license information is displayed. Click on the 'Remove License Key' button. This will walk you through the unregister process and provide you with a reference number. Write this number down!

Now, start the License Manager on your new PC. Under 'Get License Key from Web', enter the reference number you received above, along with your user name and email address (you need to use the same user name and email address you used on the first PC). Click 'Get License Key'.

Re-Installing Windows

When you re-install your Windows operating system, your PC will have a different hardware fingerprint and your original license key will no longer work.

If you need to re-install your Windows operating system, you will need to remove your license (as described above), saving the reference number before re-installing Windows. After installation, you can obtain a new license key from the License manager for your new hardware fingerprint.

Updating & Software Maintenance

You may install new versions of ARulesXL for one year after your original purchase.

If you install a version after your maintenance has expired ARulesXL WILL NOT OPERATE, and you will need to re-install your prior version.

You can renew your software maintenance annually, at which time you will be issued a new license key. To install this key, start the License Manager. Under 'Install License Key' enter your user name and license key. Then click 'Install License Key'.
