New $99 Pricing

The packaging and pricing of ARulesXL has changed from the separate developer and runtime licenses to a more flexible, single user license priced at $99. Organizations interested in multiple licenses can contact us for information on quantity discounts and site licensing.

You can purchase a copy now at the ARulesXL Web Store.

Beta Test Release Coming Soon

We are hard at work on the next release of ARulesXL. Thanks to all who provided feedback. The new release has much improved support for importing tables of data for use by the rules. These tables can be a simple list of user inputs or a sophisticated set of n-dimensional arrays. The supporting capabilities for lists and arrays makes it easier to build configuration and advanced reasoning applications.

If you have any additional suggestions for enhancements, please let us know as soon as possible. We will send out a newsletter as soon as the beta test version is available for download.
