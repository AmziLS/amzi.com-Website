Explore AI & Logic Programming

Explore and learn logic programming and artificial intelligence with Amzi!. Create logicbases that give advice, solve problems, plan, configure or play games. You access a logicbase of rules just as you would a database. The rules are expressed in Prolog which has powerful, built-in search and pattern matching capabilities.

You can deliver your logicbases in Web-based applications written in Java (JSP, Servlets) and .NET (ASP.NET, VB, C#), or in stand-alone applications written in C++, Java, Delphi, C# or VB under Windows, Linux, Solaris and HP/UX.

Amzi! features the world-class Eclipse IDE (www.eclipse.org). It provides full source code debugging, including logicbases embedded in other tools and languages or running on remote servers. The IDE has many productivity features including file outlines, project cross references, hover help and syntax assistance. Amzi! has a variety of samples and two full tutorials, one for new users, and an advanced tutorial for building specialized expert systems.
