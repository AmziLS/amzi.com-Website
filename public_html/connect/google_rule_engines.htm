Create Rule Engines for
Pricing, Workflow, Regulations,
Configuration, Scheduling and more

Create rule engines for business rules, regulations, pricing, workflow, advice, problem solving, scheduling and configuration. The rules are expressed in Prolog which has powerful, built-in search and pattern matching capabilities.

You can deliver your rule engines in Web-based applications written in Java (JSP, Servlets) and .NET (ASP.NET, VB, C#), or in stand-alone applications written in C++, Java, Delphi, C# or VB under Windows, Linux, Solaris and HP/UX.

Amzi! features the world-class Eclipse IDE (www.eclipse.org). It provides full source code debugging, including rule engines embedded in other tools and languages or running on remote servers. The IDE has many productivity features including file outlines, project cross references, hover help and syntax assistance. Amzi! has a variety of samples and two full tutorials, one for new users, and an advanced tutorial for building specialized rule engines.
