Create Business Rules in Excel Spreadsheets
Run them Anywhere

ARulesXL™ turns Microsoft® Excel® into a powerful tool for delivering business rule applications that combine decision support with computational analysis.

Instead of difficult to code, maintain and understand Excel IF statements, rules are entered directly on your spreadsheet in a natural and easy to understand syntax. The rules can then be queried, much like a database. (Click on the image below to see a simple example contrasting different approaches.)

Once created rules can be deployed in Excel, on the Web or in any application.

Embed Intelligent Logic-Bases
in .NET, Delphi, Java, C++ & more

Amzi! enables the easy integration of intelligent components with conventional applications allowing you to add logic-bases that diagnose, advise, tune, plan and parse. You access a logic-base of rules just as you would a database. The rules are expressed in Prolog which has powerful, built-in search and pattern matching capabilities.

Weather Shield provides configuration and pricing information for doors and windows (the glass type) using a hybrid logicbase. DigitalXPress schedules real-time streaming video, multicasts and large data transfers via satellites. eoTek, a supplier of mortgage services for Bank of America, maintains separate knowledgebases of pricing and underwriting rules. Two major software vendors license Amzi! products for inclusion in workflow products.

Amzi! is built on the world-class Eclipse IDE. The Amzi!® Logic Server™ is encapsulated as a Delphi Component,  Java Class (JSP and Servlets), C/C++ Class,  .NET Class (ASP.NET, VB, C#), and more. You can add your own Prolog predicates in Java, C/C++, C#, VB or Delphi. Logic-bases can reason over ODBC databases. Windows, Linux, Solaris, HP/UX.
