ASP.NET VB Pets Sample

Building the Sample

In order to run this sample you must have both IIS and Visual Studio .NET installed. Follow these steps to create the ASP.NET application:

  1. In Visual Studio, select File | New | Project. Then in the Visual Basic Projects folder, click on ASP.NET Web Application.
  2. Change the location to http://localhost/vb_pets.
  3. Compile and link pets.pro into pets.xpl.
  4. Select Project | Add Reference. Click on Browse and select amzinet.dll in the amzi/bin subdirectory. Click OK.
  5. Select File | Save All, then close Visual Studio.
  6. Copy the files from this directory to your InetPub/wwwroot/vb_pets subdirectory.
  7. Re-open Visual Studio and your vb_pets project.
  8. Select Build | Rebuild Solution.

Running the Sample

  1. Select Debug | Start to run the sample under IIS.
