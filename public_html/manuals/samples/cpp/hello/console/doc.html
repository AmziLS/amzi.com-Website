C++ Hello Console Sample

This is the simplist C++ sample that can be run from the command-line. It ensures everything is installed correctly.

Building the Sample

  1. Compile and link hello.pro into hello.xpl.
  2. Under Windows, open the Microsoft Visual C++ Project hello.dsw and build hello.exe.
  3. Under Unix, run the appropriate makefile under Linux (makefile.lnx) or Solaris (makefile.sol) to build hello

Running the Sample

  1. Run hello(.exe).
