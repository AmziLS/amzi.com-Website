Applying Logical Knowledge

  Rule Clauses

Rules can have any number of clauses, and call other rules.

cost1(FROM, TO, delta, COST) :-
   distance(FROM, TO, DIST),
   DIST < 20,
   COST is 10 * DIST.
cost1(FROM, TO, delta, COST) :-
   distance(FROM, TO, DIST),
   DIST >= 20,
   COST is 10 * DIST + 100.
cost1(FROM, TO, united, COST) :-
   distance(FROM, TO, DIST),
   COST is 8 * DIST.

NOTE - rules can fail immediately because the head doesn't unify (carrier is delta or united), or because a goal in the rule fails (distance > or < 20).

?- cost1(lax,den,delta,X).

X = 104.403 

yes
?- cost1(lax,den,C,X).

C = delta
X = 104.403 ;

C = united
X = 83.5225 ;
no
?- cost1(lax,bos,C,X).

C = delta
X = 517.732 ;

C = united
X = 334.186 ;
no
