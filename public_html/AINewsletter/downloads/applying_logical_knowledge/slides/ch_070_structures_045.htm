Applying Logical Knowledge

  Directives

Directives contain Prolog text that is called when the file is consulted, so they apply to the program. They are often used to define operators for a file.

To define an arrow (different from arithmetic >=) for routes:

:- op(500, yfx, =>).

Now => can be used in the program:

find_connect(FROM, TO, FROM => TO) :-
   flight(FROM, TO, _, _, _).
find_connect(FROM, TO, FROM => VIA => TO) :-
   flight(FROM, VIA, _, _, _),
   flight(VIA, TO, _, _, _).

And when running:

?- find_connect(lax, cvg, R).
R = lax => cvg ;
R = lax => bos => cvg ;
R = lax => den => cvg ;
no
