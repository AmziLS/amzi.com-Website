  Predicates

Predicates are one or more clauses with the same functor and number of arguments, arity. They should be contiguous. They are referred to in documentation by functor/arity.

flight/4

flight(delta, 4, 'LAX', 'DEN').
flight(delta, 5, 'BOS', 'LAX').
flight(united, 14, 'ARB', 'CVG').

vehicle/2

vehicle(43, [make = 'Suburu', model = 'Forester', color = blue]).
vehicle(44, [model = 'Miata', year = 1997, color = black]).

customer_status/2

customer_status(ID, good) :- account_balance(ID, past_due(DaysLate)), DaysLate < 30.
customer_status(ID, poor) :- account_balance(ID, past_due(DaysLate)), DaysLate >= 30.
customer_status(ID, excellent) :- owner_friend(ID).
customer_status(ID, bad) :- customer(ID, name('Dastardly', 'Dan'))
