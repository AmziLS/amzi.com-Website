Applying Logical Knowledge

  Blends

The line between logic base and logic programming is not always clear.

Many Prolog programs have a bit of both.

  • singles/birds
  • samples/prolog/configure
  • if_duck_world (contrast to nanisrch)

But the distinction helps for explaning the different, all true, notions of Prolog:

  • Prolog is easy.
  • Prolog is hard.
  • Prolog can only be used for limited types of rules.
  • Prolog can model anything.
