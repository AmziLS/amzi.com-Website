  Unification Tricks

Unification with the head can be used to pass variable bindings between argument variables.

equate(X, X).
insert(X, a(X)).
extract(a(X), X).
get_city(address(CITY, ST), CITY).

get_passenger_city(ID, CITY) :-
  passenger(_,_,ID,ADDRESS),
  get_city(ADDRESS, CITY).

Exercise - write create_address/3 where first two arguments are city and state and the third argument is an address(CITY,STATE) structure. It might be used like this:

add_passenger(FIRST, LAST, ID, CITY, STATE) :-
   create_address(CITY,STATE,ADDRESS),
   assert(passenger(LAST,FIRST,ID,ADDRESS))
