Amzi! Interactive Fiction Toolkit

The Amzi! Interactive Fiction Toolkit (AIFT) provides a framework for capitalizing on the expressiveness of executable logic in implementing interactive fiction (IF) games.

  • Overview - The July 2004 AI Expert Newsletter which discusses IF issues and a logicbase approach to the development of IF.
  • Tutorial - How to get started using the toolkit.
  • Reference - The services and features of the toolkit.

Discussions of Sample Games:

  • Cloak of Darkness - The classic small IF game used to compare authoring tools.
  • Duck World - A small game designed to illustrate various features of AIFT.

AIFT can be used as is with a text editor to create new IF games, but a Prolog development environment will provide better tools for authoring and debugging.

AIFT was implemented using Amzi! Prolog and Logic Server (www.amzi.com), but can be easily used with any ISO standard Prolog.

AIFT is a work in progress, and its current state is best described as beta. Comments and suggestions for improvements can be sent directly to the author: Dennis Merritt
