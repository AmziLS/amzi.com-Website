Downloads

These are source files and other materials that accompany some of the newsletter articles.

Applying Logical Knowledge - An online copy of the course notes for Dennis Merritt's course, Applying Logical Knowledge.

amzi_course_notes_050125.zip - course notes and samples for Dennis Merritt's course, Applying Logical Knowledge.

Amzi! IF Toolkit Documentation - An online copy of the documentation for the Amzi! Interactive Fiction Toolkit

aif_1-3.zip - The source and executables for the Duck World, Cloak of Darkness and other sample interactive fiction games. Includes the full runtime and sources and documentation. Unzip to a directory called AIFT and run duck_world.bat and or cloak.bat. This is a beta release of the Amzi! IF Toolkit so any and all comments are welcome.

fuzzy_1-1.zip - The fuzzy logic tool kit discussed in the Jan, Feb, and March 2004 newsletters.

gp_1-0.zip - Genetic programming experiments in the automatic creation of Prolog list utilities. See the May 2004 newsletter.

KASource20.zip - Full source for Army Medical Knowledge Engineering System (AMKES), a public domain pure Java 3-tiered system for encoding and reasoning over medical research results. This was the first working version of the system. An overview is available at http://www.amzi.com/articles/amkes_pajava99.htm.
