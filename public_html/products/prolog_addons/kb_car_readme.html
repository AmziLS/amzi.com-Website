Instructions for installation of The Car Identifier
on an IIS enabled personal machine

In its current form it is designed to run on an IIS machine only.

1) Install all files to the C:\inetpub\wwwroot directory, preserving the file structure that exists in the zip file.

After unzip you should have the following directories added to your wwwroot folder, also give them the access permissions (using the IIS Manager in your Control Panel) highlighted in green and application permissions highlighted in pink

/CarIdentity/images/ (46 Files) read & scripts
/CarIdentity/report_files/ (16 Files)
/CarIdentity/scripts/ (13 Files) read, script source access & Execute (includes scripts)
/CarIdentity/ (11 Files) read & Execute (includes scripts)
/database/ (2 Files)
/ (6 Files)

2) Make the cars.mdb database in /database/ a system DSN with the name "cars"

3) Run the website using http://localhost
